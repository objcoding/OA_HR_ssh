# OA-system

----------
## 本系统采用Struts2.3 + Spring4.0 + Hibernate4整合开发<br/>

### day_01<br/>
#### ssh整合的环境搭建,BaseDao设计以及部分实现类的创建,部分系统模块之岗位管理Action.<br/>

### day_02<br/>
#### 一些前端模版,系统管理模块的关系映射以及CRUD的一些改进等.<br/>

### day_03<br/>
#### 此次主要实现在增删改功能中增加对上级部门的处理,利用递归实现部门列表显示树状结构,jsp页面抽取,在显示层抽取BaseAction,合并Service层与Dao层,最后完成了岗位,员工,部门模块的基本关联关系与显示,以及懒加载和空指针的一些异常的解决.<br/>

### day_04<br/>
#### 权限模块的设计:权限实体的关联设计--->初始化权限数据--->根据权限显示模块和内容(超链接)--->使用拦截器拦截所有用户请求,根据用户的权限限定访问链接<br/>

### day_05<br/>
#### 论坛模块的开发:模块相关实体的关联映射设计,映射新的知识运用:有单向关联,表之间的多种关联,实体新增特殊属性减少数据库的查询,以及页面显示的排序的hql语句编写等.<br/>
